About the API

The API that I chose is a free dictionary which gives the definition of a word when the word is provided. In this case, I chose to display the definitions of the word "computer".


I found the API on https://dictionaryapi.dev/.


Options used for the API request: the word, "computer"


Two applications where this API is helpful: This API is helpful when a user is researching and wants the definition of a word that they come across in readings, such as "computer". It is also helpful to use when trying to expand your vocabulary and learn new words.
